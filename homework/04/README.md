# JavaScript 基礎與進階綜合練習專案

這是一個包含 10 個 JavaScript 練習的專案，旨在熟悉程式設計中的核心邏輯控制與資料結構操作。

## 涵蓋的語法與觀念
本專案的每一個練習皆經過測試，涵蓋了以下 JavaScript 核心元素：
- `if` / `else` (條件判斷)
- `for` (計次迴圈)
- `while` (條件迴圈)
- `function` (函式宣告與呼叫)
- `Array` (陣列操作)
- `Object` (物件屬性存取)
- `JSON` (`JSON.parse` 與 `JSON.stringify`)

## 練習清單
1. **條件判斷** - 檢查分數是否及格。
2. **陣列與迴圈** - 計算陣列元素的總和。
3. **未知次數的迴圈** - 模擬擲骰子直到出現 6。
4. **物件操作** - 建立包含員工資料的物件。
5. **陣列結合物件** - 篩選高薪員工名單。
6. **JSON 解析與轉換** - 修改 JSON 字串中的狀態屬性。
7. **陣列尋找** - 使用 `while` 找出陣列中的第一個負數。
8. **複雜物件走訪** - 找出系統中的管理員名單。
9. **資料重組** - 將零散陣列合併為結構化的物件陣列。
10. **終極綜合挑戰** - 解析系統 JSON 日誌，進行過濾與統計，並回傳新的 JSON 報表。

## 執行方式
1. 確認你的電腦已安裝 [Node.js](https://nodejs.org/)。
2. 將程式碼複製到副檔名為 `.js` 的檔案中（例如 `exercises.js`）。
3. 打開終端機 (Terminal) 或命令提示字元，執行以下指令即可查看輸出結果：
   ```bash
   node exercises.js
