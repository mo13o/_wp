# DC Blog — 簡易網誌系統

一個使用 Node.js + SQLite 打造的輕量網誌平台，類似 Dcard 的討論區風格。

---

## 版本介紹

### blog — 基礎版
最簡單的網誌系統，任何人都可以瀏覽、發表、編輯、刪除文章。適合單人使用或快速筆記。

### blog1 — 會員版
加入註冊與登入功能。發表、編輯、刪除文章前必須登入，且只能操作自己的文章。適合多人共用但需要區分身份的場景。

### blog2 — Dcard 風格版
全面改造 UI，採用卡片式設計與紫色主題。頂部導航欄區分「公共貼文」與「我的貼文」，讓瀏覽更直覺。

### blog3 — 個人頁面版
點擊作者名稱即可進入該用戶的個人貼文頁面，查看該用戶發表過的所有文章。

### blog4 — 板塊討論版
加入板塊分類系統（心情、男生、女生、職業、學生）。發文時可選擇所屬板塊，首頁頂部有板塊列可快速切換瀏覽。最接近 Dcard 的使用體驗。

---

## 功能一覽

- 註冊 / 登入 / 登出
- 發表、編輯、刪除文章
- 文章列表（全部、依板塊、依作者、個人）
- 板塊分類（blog4）
- 紫色卡片風格 UI
- 響應式設計（手機/電腦皆可瀏覽）

---

## 快速開始

```bash
cd blog4          # 選擇想用的版本
npm install       # 安裝依賴（首次執行）
npm start         # 啟動伺服器
```

開啟瀏覽器前往 `http://localhost:3000` 即可使用。

---

## 技術說明

- **後端**：Express.js
- **資料庫**：SQLite（better-sqlite3）
- **模板**：EJS
- **認證**：bcrypt + express-session
- **無前端框架**，純原生 CSS
