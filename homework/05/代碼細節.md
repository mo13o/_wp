# 簡易網誌系统 — 代碼細節

## 版本演進

| 目錄 | 說明 |
|------|------|
| `blog` | 基礎版 — 無認證，任何人都可 CRUD 文章 |
| `blog1` | 加入註冊/登入/登出，操作需登入且只能編輯自己的文章 |
| `blog2` | Dcard 風格 UI，新增公共貼文區與個人貼文區 (`/my`) |
| `blog3` | 作者名稱可點擊，進入該用戶的個人貼文頁面 (`/user/:id`) |
| `blog4` | 新增板塊系統（心情/男生/女生/職業/學生），發文可選板塊 |

---

## 共同技術棧

- **執行環境**：Node.js v24
- **網頁框架**：Express
- **資料庫**：SQLite（透過 `better-sqlite3`）
- **模板引擎**：EJS
- **認證**：`bcrypt` + `express-session`

---

## 資料庫結構 (`db.js`)

```sql
users
  id            INTEGER PRIMARY KEY AUTOINCREMENT
  username      TEXT NOT NULL UNIQUE
  password      TEXT NOT NULL          -- bcrypt 雜湊
  created_at    DATETIME DEFAULT CURRENT_TIMESTAMP

boards          -- 僅 blog4
  id            INTEGER PRIMARY KEY AUTOINCREMENT
  name          TEXT NOT NULL UNIQUE   -- 英文代號 (mood, male, ...)
  label         TEXT NOT NULL          -- 中文顯示 (心情, 男生, ...)

posts
  id            INTEGER PRIMARY KEY AUTOINCREMENT
  title         TEXT NOT NULL
  content       TEXT NOT NULL
  author_id     INTEGER NOT NULL       -> users.id
  board_id      INTEGER NOT NULL       -> boards.id  (blog4 新增，預設 1)
  created_at    DATETIME DEFAULT CURRENT_TIMESTAMP
  updated_at    DATETIME DEFAULT CURRENT_TIMESTAMP
```

## 路由總表

| 方法 | 路由 | 權限 | 說明 | 引入版本 |
|------|------|------|------|----------|
| GET | `/` | 公開 | 所有文章列表（全部/公共） | blog |
| GET | `/board/:id` | 公開 | 特定板塊文章列表 | blog4 |
| GET | `/my` | 需登入 | 當前用戶的文章列表 | blog2 |
| GET | `/user/:id` | 公開 | 指定用戶的文章列表 | blog3 |
| GET | `/register` | 公開 | 註冊頁面 | blog1 |
| POST | `/register` | 公開 | 註冊請求 | blog1 |
| GET | `/login` | 公開 | 登入頁面 | blog1 |
| POST | `/login` | 公開 | 登入請求 | blog1 |
| POST | `/logout` | 公開 | 登出（清除 session） | blog1 |
| GET | `/post/new` | 需登入 | 寫文章表單 | blog |
| POST | `/post/create` | 需登入 | 新增文章 | blog |
| GET | `/post/:id` | 公開 | 文章詳細頁 | blog |
| GET | `/post/:id/edit` | 需登入 + 作者 | 編輯文章表單 | blog |
| POST | `/post/:id/update` | 需登入 + 作者 | 更新文章 | blog |
| POST | `/post/:id/delete` | 需登入 + 作者 | 刪除文章 | blog |

## 中介層

```javascript
// 所有版本皆有的全域中介
app.use(express.urlencoded({ extended: true }));

// blog1+ 加入 session
app.use(session({ secret, resave: false, saveUninitialized: false }));

// blog1+ 將當前用戶注入模板
app.use((req, res, next) => {
  res.locals.user = getCurrentUser(req);
  res.locals.path = req.path;
  next();
});

// 路由保護（blog1+）
function requireAuth(req, res, next) {
  if (!req.session.userId) return res.redirect('/login');
  next();
}
```

## 啟動方式

```bash
cd <版本目錄>   # 例如 cd blog4
npm install     # 第一次執行
npm start       # 啟動於 http://localhost:3000
```

每個版本有獨立的 `package.json`，需各自執行 `npm install`。
