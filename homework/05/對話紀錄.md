# 對話紀錄

## 2026-06-12

### 1. 建立基礎網誌系統
- 使用 Node.js + Express + SQLite + EJS 建立簡易網誌
- 初始功能：CRUD 文章，無認證
- 目錄：`05/`（後續改名為 `blog`）

### 2. 修正無法顯示內容
- 舊的 `blog.db` schema 缺少 `updated_at` 欄位
- 刪除重建後正常顯示

### 3. 調整列表為「了解更多」樣式
- 列表頁改成只有標題 + 日期 +「了解更多 →」連結
- 後續應要求加回摘要內容

### 4. 加入會員系統
- 新增註冊、登入、登出功能

- 寫/編/刪文章需登入，且只能操作自己的文章
- 儲存為 `blog1`

### 5. Dcard 風格改造
- 紫色系卡片式 UI 設計
- 加入導航欄

- 儲存為 `blog2`

### 6. 個人作者頁面
- 點擊作者名稱可查看該用戶所有文章 (`/user/:id`)
- 儲存為 `blog3`

### 7. 板塊分類系統
